# Solar_System

**Solar System when you can find info about our planets, such as:**
* Distance to Sun.
* Revolution Period in Years.
* Rotation Period in Days.
* Planet Mass.
* Planet Diameter in KM.
* Average Temperature.
* Atmosphere main components.
* Moons.
